CampusLens Project Report
Problem
College clubs generate large volumes of event photos and videos, but those memories are often scattered across personal devices, Drive links, and chat groups. Students struggle to find photos from events, photographers lack a structured upload flow, and clubs cannot easily share albums or measure engagement.

Solution
CampusLens is an AI-powered event memories platform for campus organizations. It gives clubs a centralized place to create events, upload media, generate AI summaries, detect scenes and faces, share QR albums, track attendance heat, and build personalized feeds for students.

Key Modules
Event Management: event metadata, albums, timeline, heatmap
Media Management: upload intents, gallery, likes, favorites, duplicate detection
AI Layer: scene detection, face clustering, captions, smart search, recommendations
Social Layer: comments-ready schema, likes, notifications, personalized memory feed
Security Layer: JWT-style auth, RBAC, rate limiting, audit logs
Deployment Layer: Docker, environment config, CI workflow
Innovation
CampusLens goes beyond a normal gallery by treating event albums as living memory systems. The product ranks club impact, generates event briefs, identifies duplicate/low-quality media, recommends similar images, and curates personal collections based on face clusters and visual scenes.

Resume-Ready Summary
Designed and developed an AI-powered media management ecosystem for campus organizations, enabling event-wise albums, AI-generated memory highlights, personalized face-based discovery, QR album sharing, role-based access, and production-ready deployment workflows.
